Trivia Quiz

Trivia Quiz Project utilizes HTML, CSS, and JavaScript to fetch a series of true/false questions from the Open Trivia Database API and present them sequentially to users. JavaScript handles fetching data asynchronously, manipulating the DOM to display questions, and providing immediate feedback on user responses. The project's frontend, composed of HTML for structure and CSS for styling, ensures a user-friendly interface with responsive design elements. Essential skills for this project include API integration, asynchronous programming, DOM manipulation, and frontend development techniques. Through these components, the quiz application delivers an interactive experience where users can test their knowledge and track their performance in real-time.

Routes API

This project is a Flask-based REST API designed to help users find the most cost-effective route between two locations by intelligently integrating with the Google Maps API and a custom dataset of gas stations. The core functionality analyzes available fuel stops along a given route and dynamically selects the optimal ones based on profitability, distance, and fuel prices. It leverages geospatial calculations, external API integration, and custom business logic to deliver optimized travel plans for drivers looking to minimize fuel costs. This solution is particularly useful for logistics companies, long-haul drivers, or budget-conscious travelers.

Technologies used include Flask for backend API development, Google Maps API for geolocation and routing, and custom Python scripts for data processing and optimization. The project emphasizes practical use of API composition, route planning, and efficient data modeling.
